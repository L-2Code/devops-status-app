# DevOps Status App

A small DevOps demo project showcasing containerization and CI/CD fundamentals.

## Features
- Node.js/Express backend
- Static frontend served via Nginx
- Dockerized multi-container setup
- CI pipeline with GitHub Actions

## Tech Stack
- Node.js
- Express
- Docker
- Docker Compose
- Nginx
- GitHub Actions

## Run locally
```bash
docker compose up --build
